# Pool Settings FAQ — Pick the Game Pools

### My pool switched to Confidence picks — can I change it back to Pick’em?
**Tags:** pool-settings, confidence, pickem, configuration

Pool type (Pick’em vs Confidence) is set in pool settings.  
If the option is locked after creation, contact support and it can be adjusted for you.

### Can submission schedules be changed during the season?
**Tags:** submission, schedule, pool-settings, deadlines

Yes. Commissioners can modify submission timing anytime:

**Pool Settings → Submission Schedule**

Options include:

- Before all games  
- Before Sunday games  
- Before each game
